1.- Definición de HTML
Lenguaje de Marcas de Hipertexto (HTML por sus siglas en inglés), es un lenguaje de marcas para programar páginas web que combina etiquetas HTML, propiedades CSS, Javascript.

2.- Etiquetas Básicas
HTML tiene las siguientes etiquetas básicas:

  • body: para el contenido
  • head: para información sobre el documento
  • div: división dentro el contenido
  • a: para enlaces
  • strong: para poner le texto en negrita
  • br: para saltos de línea
  • h1-h6: para títulos dentro del contenido
  • img: para insertar imágenes al contenido
  • p: para parágrafos

3.- Los componentes y las etiquetas
Un documento HTML está compuesto en su gran mayoría por texto, pero también puede contener listas, tablas, enlaces, gráficos, imágenes, sonidos, videos, animaciones, entre otros. Cada uno se define a través de una etiqueta o marca específica.
