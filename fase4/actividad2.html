Temas

La temática que se abordará en este sitio web está relacionado con el diseño de sitios web, como las etiquetas básicas de HTML, atributos y valores, CSS Grid y Flexbox.

Etiquetas Básicas de HTML

El Lenguaje de Marcas de Hipertexto tiene las siguienes etiquetas básicas, body para el contenido, head para información sobre el documento, div para división dentro del contenido, (a) para insertar enlaces, (br) para saltos de línea, (strong) para poner el texto en negrita, (h1...h6) para títulos dentro del contenido, (img) para insertar imágenes al contenido, (p) para párrafos.

Atributos en HTML

Los atributos permiten personalizar las etiquetas de HTML, los atributos comunes se dividen en: atributos básicos que se pueden utilizar en casi todas las etiquetas, atributos de internacionalización, se utilizan en páginas que contienen información en varios idiomas, atributos de eventos, se utilizan en páginas creadas con JavaScript y atributos de foco, que está relacionados con la accesibilidad de los sitios web.

CSS Grid

CSS Grid Layout contiene funciones de diseño dirigidas a desarrolladores de aplicaciones web, permite que las páginas web se visualicen de manera correcta en cualquier pantalla. CSS Grid es una herramienta fundamental para crear estructuras de sitios web de una forma mucho más fácil, permitiendo tener el control de toda la estructura, para decidir la distribución del sitio.

Flexbox

Flexbox es un modelo de diseño CSS, permite mejorar el diseño responsive y darle estructura a una página web de manera más fácil a la tradicional, evitando el uso de float y propiedades similares, escribiendo menos líneas de código y facilitando el posicionamiento de los elementos. Con el uso de Flexbox podremos alinear, dar tamaño y distribuir el espacio del sitio web.
